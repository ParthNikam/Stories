## Getting Started
Follow these steps to run the project:

1. Open a terminal and navigate to the server directory:
```code
cd server
```

2. Start the server:
```code
npm run start
```

3. Open another terminal and navigate to the client directory:
```code
cd client
```

4. Start the client:
```code
npm start
```
